Baby Monitor

Demo

The Baby Monitor System, a childcare innovation that empowers caregivers with real-time insights into infants' well-being, even from afar. This system employs cutting-edge sensors and intelligent monitoring to ensure utmost vigilance. It features a sensitive motion detector, a calibrated force sensor for crib presence, and a precision microphone for capturing sounds. Visual feedback via an LED matrix reassures with smiles or alerts with frowns, while a user-friendly joystick enables sound adjustments. Powered by BeagleBone Green, the Baby Monitor seamlessly integrates technology and design to redefine infant care, fostering trust in distant caregiving.

Beat Box

Demo

Beat-Box is a sophisticated endeavor in embedded systems, focusing on audio generation and hardware control. Built on the BeagleBone platform, it transforms this device into a dynamic digital drum kit. Central to the project is real-time audio generation through the ALSA API, with multi-threaded execution.

Beat-Box offers diverse drum beat modes, customizable sounds, and precise control over ProjectListo and volume parameters. Hardware components, including a joystick, LED matrix, and buttons, enhance user interaction. The project prioritizes memory management and Valgrind analysis for stability.

This technical playground invites enthusiasts to explore embedded audio processing, hardware integration, and real-time optimization. Whether you're a seasoned developer or a curious learner, Beat-Box presents a captivating journey into the world of embedded audio systems.

Trackour

Demo

Trackour is a web application that assists both beginner and experienced music creators with planning their mix or mashup projects. Users will be able to search for music through Trackour using harmonic details of the songs, such as key, ProjectListo, time signature, etcetera. The search function will give users the option to narrow their search down to harmonically compatible songs, which will be determined using a system based on the Camelot Wheel. Additionally, Trackour will provide collaborative functionality, allowing users to plan, save, and share their mix projects with other users.

Histogram

Demo

Histogram Generator is a web-page that gives the count of each student in their grade categories. This page incorporates input boxes for grades, expertly managed by a JavaScript file, "calculate.js." Visitors to the website can input new grades, and may also change the grade boundaries as they wish, histogram dynamically adapts to illustrate the distribution of students across various grade ranges. To ensure a seamless user experience, the system gracefully handles invalid inputs, providing helpful feedback.
